# react-fastapi-payroll-API
Working In Progress

### Instruction 
python3 -m venv .env
source .env/bin/activate

### Install requires
pip3 install requirements.txt

### Install react dependencies
cd ./frontend/src && npm install

### Start app
### Run api
cd ./backend && uvicorn main:app -- reload

### Run react
npm start
